# User Student Management System

This project is a **full-stack Student Management System** developed using  
**ASP.NET Core Web API (.NET 8)** for the backend and **Angular (Standalone Components)** for the frontend.

The application allows users to **register and log in**, and after authentication, perform **CRUD operations on students**.
